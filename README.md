# Livora — Landing

Landing page de **Livora**, la plataforma financiera y operativa hecha a la medida de los franquiciatarios de Farmacias Similares.

Sitio estático de un solo archivo (`index.html`) con imágenes en `assets/`.

## Desarrollo

No requiere build. Abre `index.html` en el navegador o sirve la carpeta:

```bash
python3 -m http.server 5173
```

## Deploy

Conectado a **Vercel** vía GitHub: cada push a `main` dispara un deploy automático.
